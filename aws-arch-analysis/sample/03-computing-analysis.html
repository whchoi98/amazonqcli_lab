🔧 최적화 기회:
- EKS 노드 Auto Scaling 활용
- Spot Instance 도입 검토

3. 워크로드 VPC (VPC01, VPC02)

✅ 일관된 구성:
- 표준화된 인스턴스 타입
- 균등한 워크로드 분산
- 예측 가능한 성능

📈 확장 계획:
- 워크로드 증가 시 Auto Scaling 그룹 활용
- 성능 요구사항에 따른 인스턴스 타입 조정
