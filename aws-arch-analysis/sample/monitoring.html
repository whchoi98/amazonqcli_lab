📋 권장 대시보드 구성

📊 통합 모니터링 대시보드

1. 인프라 대시보드

  • EC2 메트릭: CPU, 메모리, 디스크, 네트워크
  • EBS 메트릭: IOPS, 처리량, 대기 시간
  • 네트워크 메트릭: VPC Flow, NAT Gateway
  • 비용 메트릭: 일일/월간 비용 추이

2. 애플리케이션 대시보드

  • EKS 메트릭: Pod, 노드, 서비스 상태
  • 컨테이너 메트릭: CPU, 메모리, 네트워크
  • 애플리케이션 메트릭: 응답 시간, 에러율
  • 비즈니스 메트릭: 사용자 활동, 트랜잭션

3. 보안 대시보드

  • 보안 이벤트: CloudTrail, GuardDuty
  • 네트워크 보안: VPC Flow Logs 분석
  • 컴플라이언스: Config Rules 상태
  • 액세스 패턴: 비정상 로그인 탐지
